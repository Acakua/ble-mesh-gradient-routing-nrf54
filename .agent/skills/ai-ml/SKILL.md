# AI/ML Workflow Bundle

## Overview
Workflow for building LLM applications, RAG systems, AI agents, and ML pipelines.

## When to Use This Workflow
- Building LLM-powered applications.
- Implementing RAG.
- Creating AI agents.
- Developing ML pipelines.

## Workflow Phases
1. **Design**: Define use cases, choose models.
2. **Integration**: Prompt engineering, API setup.
3. **RAG**: Data pipeline, vector database, retrieval.
4. **Agents**: Roles, tools, orchestration.
5. **ML Pipeline**: Data processing, training, evaluation.
6. **Observability**: Tracing, logging, evaluation.
7. **Security**: Validation, filtering, audit.

## Quality Gates
- All AI features tested.
- Performance benchmarks met.
- Security and observability configured.
